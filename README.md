# Enquiry Management System

A modern full-stack Enquiry Management System developed using Angular and ASP.NET Core Web API. The application allows users to submit enquiries online while providing administrators with a secure JWT-based dashboard to manage services and customer enquiries.

##  Features

### User Module
- Submit enquiries online
- View available services
- Contact page
- Responsive user interface
- Client-side form validation

### Admin Module
- Secure Admin Login
- JWT Authentication
- Manage Services (CRUD)
- Manage Customer Enquiries
- Protected API Endpoints
- Dashboard

##  Installation

### Backend

1. Open the ASP.NET Core project in Visual Studio.
2. Update the SQL Server connection string in `appsettings.json`.
3. Apply Entity Framework migrations.
4. Run the API.

### Frontend

1. Open the Angular project in Visual Studio Code.
2. Install dependencies:

```bash
npm install
```

3. Run the application:

```bash
ng serve
```

4. Open:

```
http://localhost:4200
```
